# `12_template_sensors/` — Point d'entrée

---

## Rôle de ce dossier

`12_template_sensors/` contient les entités `sensor` / `binary_sensor` **template**
(couche de perception / décision). C'est du **code runtime**, pas de la doctrine.

## Avant de modifier

Ce fichier est un simple **pointeur de découvrabilité**, non normatif : il ne reproduit
aucune règle. En cas de divergence, la documentation centrale fait foi. Avant de créer
ou modifier un template sensor, lire le document de structure applicable — ne jamais se
fonder sur un fichier voisin.

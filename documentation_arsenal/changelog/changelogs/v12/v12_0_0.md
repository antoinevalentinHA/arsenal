## Verdict

v12 est une version de **durcissement architectural majeur**.

- L'exécution directe sans admissibilité est désormais interdite.
- Les helpers ne sont plus tolérés comme sources de vérité.
- La vérité physique et les capteurs contractuels deviennent les seules sources exploitables.
- Les domaines météo basculent vers des pipelines contractuels explicites.
- Les états système critiques introduisent des capteurs d'intégrité formels.

Cette version marque une transition claire : d'un système fonctionnel vers un système **opposable, vérifiable et défensif**.

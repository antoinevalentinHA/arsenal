# 🔒 AÉRATION — ROBUSTESSE MÉTIER

## 1️⃣ Pipeline — Fin d’épisode (M2) blindée

### Sécurisation forte :

- Double trigger fermeture :
  - `binary_sensor.fenetre_ouverte_maison`
  - `binary_sensor.fenetre_ouverte_maison_avec_delai`

- Condition explicite sur le `trigger.entity_id`
- Exigence `for:` sur capteur avec délai

### Guard dur ajouté dans `m2_fin_episode` :

Refus explicite si :
- Une ouverture brute est encore active
- OU une ouverture qualifiée est encore active

➡ Logbook explicite en cas de refus  
➡ Stop immédiat sans effet de bord  

Conséquence :
> M2 ne peut plus s’exécuter dans un état incohérent.

## 2. Décision Centrale — Intégration normative du pré-confort Vacances

**`10_scripts/chauffage/decision_centrale.yaml`**

Le niveau Vacances devient conditionnel :

```
Vacances + pre_confort_actif_calcule == on  →  Confort
Vacances sinon                              →  Réduit
```

- Nouvelle raison métier émise : `pre_confort_vacances`
- Le pré-confort constitue la seule exception contractuellement explicite au comportement standard Vacances
- Aucune modification de la hiérarchie globale ; l'hystérésis décisionnelle est préservée

Convention généralisée : suppression des états intermédiaires, standardisation binaire `alert` / `off`.

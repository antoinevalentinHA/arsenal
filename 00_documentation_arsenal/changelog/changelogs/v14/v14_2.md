# Arsenal — Changelog v14.2

## Cadre

v14.2 est une version de gouvernance documentaire ciblée.

Elle ne modifie aucun comportement Home Assistant :
- aucun capteur ;
- aucune automatisation ;
- aucun script ;
- aucun helper ;
- aucune logique Lovelace.

Le périmètre est exclusivement documentaire : recentrage des outils externes autour du
NAS Arsenal et suppression de l'ancienne documentation mobile / PC devenue obsolète.
